# Football Match Predictor

Predicts the outcome of football matches (Home Win / Away Win / Draw) using machine learning.

## Dataset
Kaggle European Soccer Database — 25,000+ matches across 11 European leagues.

## How it works
- Uses betting odds from Bet365 and Betway as features
- Trained a Random Forest Classifier (100 trees)
- Achieved 45% accuracy on test data (random baseline is 33%)

## Tech Stack
- Python
- pandas, scikit-learn

## Run the project
pip install pandas scikit-learn
python test.py
